Welcome!

I’m Bobby Corey, a passionate junior developer completing my Associate’s Degree in Computer Information Systems and Computer Programming at CNM. My work focuses on designing and building interactive applications and games using technologies such as C#, C++, Java, Python, and JavaScript.

I have hands-on experience with Unity and Unreal Engine, and enjoy creating both 2D and 3D games that challenge my creativity and problem-solving skills. I also use Blender for 3D modeling and GitHub for version control and collaboration.

This portfolio highlights some of my favorite programming and game development projects. Whether it’s a browser-based game or a Unity-powered prototype, each project reflects my dedication to growth and learning.

Thank you for visiting! I’m eager to contribute to innovative teams, especially in software development, game design, or QA roles. Let’s build something great together.
